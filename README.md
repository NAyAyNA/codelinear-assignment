
# CodeLinear Assignment

A responsive landing page built with React, Vite, and CSS based on the provided Figma design.

## Tech Stack

- React
- Vite
- CSS3

## Deployment

The project is deployed using Vercel.
Link: https://codelinear-assignment-phi.vercel.app/

## Project Setup

1. Clone the repository

```bash
git clone <repository-url>
cd codelinear-assignment
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```

4. Open the application in your browser

```txt
http://localhost:5173
```

## Build for Production

```bash
npm run build
```
